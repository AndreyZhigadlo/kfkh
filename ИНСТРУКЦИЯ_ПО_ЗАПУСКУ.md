# 🐄 Система учета КРС PRO - Инструкция по установке и запуску

## ❗ ПРОБЛЕМА С ВАШЕЙ СИСТЕМОЙ

У вас установлена **нестандартная версия Python 3.14** (pythoncore), которая имеет проблемы с совместимостью PyQt5 и PyInstaller.

## ✅ РЕШЕНИЕ - 3 ВАРИАНТА

### Вариант 1: Быстрый запуск (РЕКОМЕНДУЕТСЯ)

1. **Скачайте и установите официальный Python 3.12:**
   - Перейдите на https://www.python.org/downloads/release/python-3129/
   - Скачайте "Windows installer (64-bit)"
   - **ВАЖНО:** При установке отметьте галочку ☑️ **"Add Python to PATH"**

2. **Запустите установку библиотек:**
   ```
   Установка_КРС.bat
   ```

3. **Запустите программу:**
   ```
   Запустить_КРС.bat
   ```

---

### Вариант 2: Ручная установка (если Вариант 1 не помог)

Откройте командную строку (cmd) и выполните:

```cmd
REM 1. Удалите старую версию PyQt5
pip uninstall PyQt5 PyQt5-Qt5 PyQt5-sip -y

REM 2. Переустановите PyQt5
pip install PyQt5==5.15.9 --force-reinstall

REM 3. Проверьте запуск программы
python main.py
```

Если появилась ошибка про "Qt platform plugin", установите переменную окружения:

```cmd
set QT_QPA_PLATFORM_PLUGIN_PATH=C:\Users\Гульнара\AppData\Local\Python\pythoncore-3.14-64\Lib\site-packages\PyQt5\Qt5\plugins\platforms
python main.py
```

---

### Вариант 3: Создание EXE файла (только после установки официального Python 3.12!)

После установки официального Python 3.12:

```cmd
REM 1. Установите PyInstaller
pip install pyinstaller

REM 2. Создайте EXE файл
pyinstaller --noconfirm --onefile --windowed --name "KRS_Uchet_PRO" ^
  --add-data "krs_data.db;." ^
  --hidden-import=PyQt5.QtSvg ^
  --hidden-import=PyQt5.QtWidgets ^
  main.py
```

Готовый EXE файл будет в папке `dist\KRS_Uchet_PRO.exe`

## ❓ Частые проблемы

### Ошибка: "Qt platform plugin not found"
**Решение:** Установите официальный Python 3.12 вместо pythoncore

### Ошибка: "PyInstaller не найден"
**Решение:** 
```cmd
pip install pyinstaller
```

### Ошибка: "ModuleNotFoundError: No module named 'PyQt5'"
**Решение:**
```cmd
pip install PyQt5==5.15.9 --force-reinstall
```
