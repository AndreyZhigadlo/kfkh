# 🐮 Система учета КРС PRO v2.0

Профессиональная программа для учета крупного рогатого скота (КРС).

## ✨ Возможности программы:

### 📋 Основные модули:
1. **🐮 Картотека** - полная база данных животных
   - Учет бирок и чипов
   - Родословная (мать/отец)
   - Группы и статусы
   - Фотографии (опционально)
   - Вес животных

2. **💊 Здоровье** - ветеринарный учет
   - Вакцинации
   - Лечение
   - Осмотры
   - Препараты и сроки действия

3. **⚖️ Взвешивание** - контроль веса
   - История взвешиваний
   - Динамика привеса
   - Примечания

4. **🧬 Воспроизводство** - племенной учет
   - Осеменение
   - Отелы
   - Запуск
   - Учет быков-производителей

5. **📊 Дашборд** - статистика хозяйства
   - Общее поголовье
   - Количество коров и телят
   - Статистика вакцинаций

## 🚀 Установка и запуск:

### Вариант 1: Запуск из исходного кода
```bash
pip install PyQt5
python main.py
```

### Вариант 2: Создание EXE файла (Windows)
```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name="Учет_КРС_PRO" main.py
```

Или использовать spec файл:
```bash
pyinstaller Учет_КРС.spec
```

Готовый EXE файл появится в папке `dist/`.

## 💾 База данных:
- SQLite база данных `krs_data.db` создается автоматически
- При использовании EXE файла БД хранится рядом с исполняемым файлом
- Автосохранение каждые 30 секунд

## 🎨 Особенности:
- Современный интерфейс в стиле Windows 10/11
- Excel-подобная фильтрация по столбцам
- Сортировка по любому полю
- Быстрый поиск по всей базе
- Поддержка различных форматов дат

## 📝 Системные требования:
